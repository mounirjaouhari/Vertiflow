# AI Writing Prompt System for Scientific Article on VertiFlow

## Purpose
This prompt system is designed to guide a domain-specialized AI in writing a comprehensive scientific article about the VertiFlow project, focusing on automated basil cultivation in vertical farming.

---

## General Instructions
- Write in clear, academic English suitable for a peer-reviewed journal.
- Structure the article according to the provided outline.
- Use precise technical vocabulary relevant to smart agriculture, IoT, data engineering, and machine learning.
- Support claims with data, references, or logical reasoning.
- Ensure each section is self-contained but contributes to the overall narrative.

---

## Section-by-Section Prompts

### 1. Introduction
- Explain the context and importance of vertical farming.
- Describe the challenges of basil cultivation in controlled environments.
- State the main objectives and innovations of the VertiFlow project.

### 2. Related Work
- Summarize existing smart agriculture platforms and their limitations.
- Highlight what differentiates VertiFlow from previous solutions.

- Detail the hardware setup: sensors, actuators, network infrastructure.

- Outline data validation and governance protocols.

- Describe the machine learning models (Oracle ML) and their application.
- Present the bio-physical simulator and its integration.
- Discuss how external data is incorporated into the system.
